About PlantDoc
Open Source Plant Disease Detection

🌿 PlantDoc

PlantDoc is a free, open-source plant disease detection system built with TensorFlow and FastAPI. It helps farmers, gardeners, and researchers quickly identify plant diseases from leaf photos.

  • 🧠Model: TensorFlow CNN (38+ disease classes)
  • 🛡️Gatekeeper: YOLOv8 leaf detector (rejects non-leaf images)
  • ⚙️Backend: FastAPI (Python)
  • 🎨Frontend: Pure HTML / CSS / JS
  • 📄License: MIT — Free to use, modify, and distribute
  • 🔒No login required · No data stored · No tracking
